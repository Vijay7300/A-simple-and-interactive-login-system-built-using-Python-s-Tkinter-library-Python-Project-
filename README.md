# GUI Login Form using Python (Tkinter)

##  Overview
This project is a **GUI-based Login Form** developed using **Tkinter**. It allows users to enter their email and password and validates credentials with a simple authentication logic.
The application provides instant feedback using pop-up messages for successful or failed login attempts.


##  Features
-  User-friendly GUI interface  
-  Basic authentication system  
-  Instant login feedback using message boxes  
-  Styled layout with colors and fonts  
-  Lightweight and easy to run  

## Technologies Used
- Python  
- Tkinter (GUI)  
- PIL (Python Imaging Library)
